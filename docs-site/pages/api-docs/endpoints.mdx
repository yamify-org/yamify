# API Endpoints

This page documents all available API endpoints in the Yamify platform. Each endpoint includes information about request methods, URL parameters, request body format, response format, and example usage.

## Authentication

Most API endpoints require authentication. See the [Authentication](/api-docs/authentication) page for details on how to authenticate with the API.

## Base URL

All API URLs referenced in this documentation have the following base:

```
https://api.yamify.com/v1
```

For local development:

```
http://localhost:3000/api/v1
```

## User Endpoints

### Get Current User

Retrieve information about the authenticated user.

```http
GET /users/me
```

#### Response

```json
{
  "status": "success",
  "data": {
    "user": {
      "id": "usr_12345",
      "username": "johndoe",

      "name": "John Doe",
```

## Project Endpoints

### List Projects

Retrieve a list of projects for the authenticated user.

```http
GET /projects
```

#### Query Parameters

| Name | Type | Description | Default |
|------|------|-------------|---------|
| page | number | Page number | 1 |
| limit | number | Number of items per page | 20 |
| status | string | Filter by status (active, archived, deleted) | active |
| sortBy | string | Field to sort by | createdAt |
| sortOrder | string | Sort direction (asc, desc) | desc |

#### Response

```json
{
  "status": "success",
  "data": {
    "projects": [
      {
        "id": "prj_67890",
        "name": "Marketing Website Redesign",
        "description": "Project to redesign the company marketing website",
        "ownerId": "usr_12345",
        "collaborators": ["usr_54321", "usr_98765"],
        "status": "active",
        "settings": {
          "theme": "light",
          "visibility": "private",
          "notifications": true
        },
        "createdAt": "2023-06-01T10:00:00Z",
        "updatedAt": "2023-06-15T16:45:20Z"
      },
      // Additional projects...
    ]
  },
  "pagination": {
    "current_page": 1,
    "total_pages": 3,
    "total_items": 42,
    "items_per_page": 20
  }
}
```

## Error Codes

The API uses the following error codes:

| Code | Description |
|------|-------------|
| UNAUTHORIZED | Authentication is required or has failed |
| FORBIDDEN | The authenticated user does not have permission |
| NOT_FOUND | The requested resource was not found |
| VALIDATION_ERROR | The request contains invalid data |
| RATE_LIMIT_EXCEEDED | Too many requests have been made |
| INTERNAL_SERVER_ERROR | An unexpected error occurred on the server |

## Rate Limiting

The API enforces rate limits to prevent abuse. Current limits are:

- 100 requests per minute for authenticated users
- 20 requests per minute for unauthenticated users

Rate limit information is included in the response headers:

```
X-RateLimit-Limit: 100
X-RateLimit-Remaining: 99
X-RateLimit-Reset: 1625097601
```

## Versioning

API versioning is done through the URL path. The current version is `v1`. When we make backwards-incompatible changes, we will increment the version number.

```
https://api.yamify.com/v1/resource
```

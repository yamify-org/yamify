# Contact Us

We're here to help with any questions or issues you may have regarding Yamify.

## Support Channels

### GitHub Issues

For bug reports, feature requests, or technical issues, please use our GitHub issues tracker:

[https://github.com/yamify-org/yamify/issues](https://github.com/yamify-org/yamify/issues)

Please follow the issue templates provided to ensure we have all the information needed to address your concern.

### Email Support

For general inquiries, you can reach our support team at:

**support@yamify.com**

We typically respond within 24-48 business hours.

### Community Discussions

Join our community discussions on GitHub Discussions or Discord:

- [GitHub Discussions](https://github.com/yamify-org/yamify/discussions)
- [Discord Community](https://discord.gg/yamify)

## Reporting Security Vulnerabilities

If you discover a security vulnerability within Yamify, please send an email to **security@yamify.com**. All security vulnerabilities will be promptly addressed.

Please do not disclose security vulnerabilities publicly until they have been handled by our team.

## Feature Requests

We welcome suggestions for new features or improvements. To submit a feature request:

1. Check if the feature has already been requested in our GitHub issues
2. If not, create a new issue with the "Feature Request" template
3. Provide a clear description of the feature and why it would be valuable

## Contributing

Interested in contributing to Yamify? Check out our [Contributing Guidelines](/development/contributing) to get started.

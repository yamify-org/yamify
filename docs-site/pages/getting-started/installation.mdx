# Installation

This guide will walk you through the process of setting up Yamify for local development.

## Prerequisites

Before installing Yamify, ensure you have the following installed on your system:

- Node.js (v18.0.0 or later)
- npm (v8.0.0 or later) or yarn (v1.22.0 or later)
- Git

## Clone the Repository

```bash
git clone https://github.com/yamify-org/yamify.git
cd yamify
```

## Install Dependencies

Using npm:

```bash
npm install
```

Or if you prefer yarn:

```bash
yarn install
```

## Environment Setup

1. Create a `.env.local` file in the root directory of the project
2. Copy the contents from `.env.example` to `.env.local`
3. Update the environment variables as needed

```bash
cp .env.example .env.local
```

## Start the Development Server

```bash
npm run dev
```

Or with yarn:

```bash
yarn dev
```

The application should now be running at [http://localhost:3000](http://localhost:3000).

## Building for Production

To create a production build:

```bash
npm run build
npm run start
```

Or with yarn:

```bash
yarn build
yarn start
```

## Troubleshooting

If you encounter any issues during installation, check the following:

- Ensure you have the correct Node.js version installed
- Check if all dependencies were installed correctly
- Verify that your environment variables are set properly

If problems persist, please [raise an issue](https://github.com/yamify-org/yamify/issues) on our GitHub repository or check the [support section](/support/contact) for additional help.

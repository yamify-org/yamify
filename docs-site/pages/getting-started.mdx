# Getting Started with Yamify

Welcome to Yamify! This guide will help you get up and running with Yamify in your project. Follow these steps to install Yamify, understand its core concepts, and build your first feature.

## Prerequisites

Before you begin, make sure you have the following installed:

- Node.js (v18.0.0 or later)
- npm (v8.0.0 or later) or yarn (v1.22.0 or later)
- Git

## Installation

### Creating a New Yamify Project

The easiest way to get started is using the Yamify CLI to create a new project:

```bash
npx create-yamify-app my-yamify-project
cd my-yamify-project
```

This will set up a new Yamify project with all the necessary dependencies and configuration.

### Adding Yamify to an Existing Project

If you want to add Yamify to an existing Next.js project, you can install the required packages:

```bash
npm install @yamify/core @yamify/ui @yamify/hooks
# or
yarn add @yamify/core @yamify/ui @yamify/hooks
```

Then, add the Yamify provider to your application:

```tsx
// In _app.tsx or app/layout.tsx
import { YamifyProvider } from '@yamify/core';

export default function App({ Component, pageProps }) {
  return (
    <YamifyProvider>
      <Component {...pageProps} />
    </YamifyProvider>
  );
}
```

## Project Structure

A typical Yamify project has the following structure:

```
my-yamify-project/
├── public/              # Static files
├── src/
│   ├── app/             # App Router pages and layouts
│   ├── components/       # Reusable components
│   │   ├── ui/          # UI components
│   │   └── layout/      # Layout components
│   ├── hooks/           # Custom React hooks
│   ├── lib/             # Utility functions and libraries
│   ├── services/        # API services
│   └── theme/           # Theme configuration
├── .eslintrc.js         # ESLint configuration
├── .gitignore           # Git ignore file
├── next.config.js       # Next.js configuration
├── package.json         # Project dependencies
├── tailwind.config.js   # TailwindCSS configuration
├── tsconfig.json        # TypeScript configuration
└── README.md            # Project documentation
```

## Core Concepts

### 1. Component System

Yamify is built around a component-based architecture. Components are organized into these categories:

- **UI Components**: Buttons, cards, inputs, etc.
- **Layout Components**: Containers, grids, flexbox layouts
- **Form Components**: Form controls, inputs, selects, etc.

See the [UI Components](/components/ui-components), [Layout Components](/components/layout-components), and [Form Components](/components/form-components) documentation for details.

### 2. Authentication

Yamify provides a complete authentication system with JWT support, OAuth integration, and secure token management. Learn more in the [Authentication](/api-docs/authentication) documentation.

### 3. API Integration

Yamify includes utilities for interacting with backend APIs, including request handling, error management, and data fetching. See the [API Endpoints](/api-docs/endpoints) documentation.

### 4. Theming

Customize the look and feel of your Yamify application using the powerful theming system. Learn more in the [Theming](/development/theming) documentation.

## Your First Yamify Page

Let's create a simple page that displays a list of projects:

```tsx
// src/app/projects/page.tsx
import { Card, Heading, Text, Button } from '@yamify/ui';
import { Container, Grid } from '@yamify/ui/layout';
import { useProjects } from '@/hooks/useProjects';

export default function ProjectsPage() {
  const { projects, loading, error } = useProjects();
  
  if (loading) return <div>Loading projects...</div>;
  if (error) return <div>Error loading projects: {error.message}</div>;
  
  return (
    <Container>
      <div className="py-6">
        <div className="flex justify-between items-center mb-6">
          <Heading level={1}>My Projects</Heading>
          <Button variant="solid" colorScheme="primary">New Project</Button>
        </div>
        
        <Grid columns={{ base: 1, md: 2, lg: 3 }} gap={6}>
          {projects.map((project) => (
            <Card key={project.id}>
              <Heading level={3}>{project.name}</Heading>
              <Text className="mt-2 text-gray-600">{project.description}</Text>
              <div className="mt-4">
                <Button variant="outline" size="sm">View Details</Button>
              </div>
            </Card>
          ))}
        </Grid>
      </div>
    </Container>
  );
}
```

## Authentication Example

Here's how to implement a login form using Yamify components:

```tsx
// src/app/login/page.tsx
import { useState } from 'react';
import { useRouter } from 'next/navigation';
import { Card, Heading, Button, Input, FormControl, FormLabel } from '@yamify/ui';
import { useAuth } from '@/hooks/useAuth';

export default function LoginPage() {
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');
  const [error, setError] = useState('');
  const { login } = useAuth();
  const router = useRouter();
  
  const handleSubmit = async (e) => {
    e.preventDefault();
    setError('');
    
    try {
      await login(email, password);
      router.push('/dashboard');
    } catch (err) {
      setError('Invalid email or password');
    }
  };
  
  return (
    <div className="flex justify-center items-center min-h-screen bg-gray-50">
      <Card className="w-full max-w-md p-8">
        <Heading level={2} className="mb-6 text-center">Log in to Yamify</Heading>
        
        {error && (
          <div className="mb-4 p-3 bg-error-50 text-error-700 rounded">
            {error}
          </div>
        )}
        
        <form onSubmit={handleSubmit}>
          <FormControl className="mb-4">
            <FormLabel htmlFor="email">Email</FormLabel>
            <Input 
              id="email"
              type="email" 
              value={email} 
              onChange={(e) => setEmail(e.target.value)} 
              required 
            />
          </FormControl>
          
          <FormControl className="mb-6">
            <FormLabel htmlFor="password">Password</FormLabel>
            <Input 
              id="password"
              type="password" 
              value={password} 
              onChange={(e) => setPassword(e.target.value)} 
              required 
            />
          </FormControl>
          
          <Button type="submit" variant="solid" colorScheme="primary" className="w-full">
            Log In
          </Button>
        </form>
      </Card>
    </div>
  );
}
```

## API Integration Example

Here's how to fetch data from the Yamify API:

```tsx
// src/hooks/useProjects.ts
import { useState, useEffect } from 'react';
import { fetchApi } from '@yamify/core';

export function useProjects() {
  const [projects, setProjects] = useState([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);
  
  useEffect(() => {
    async function loadProjects() {
      try {
        setLoading(true);
        const response = await fetchApi('/projects');
        setProjects(response.data.projects);
        setError(null);
      } catch (err) {
        setError(err);
      } finally {
        setLoading(false);
      }
    }
    
    loadProjects();
  }, []);
  
  return { projects, loading, error };
}
```

## Troubleshooting

### Common Issues

#### "Module not found" errors

If you encounter module not found errors, make sure all dependencies are properly installed:

```bash
npm install
# or
yarn install
```

Also check your import paths and ensure they match your project structure.

#### Styling issues

If your components don't look as expected, ensure TailwindCSS is properly configured:

1. Check that your `tailwind.config.js` file is correctly set up
2. Make sure the Tailwind directives are included in your CSS
3. Verify that the `@yamify/ui` package is correctly imported

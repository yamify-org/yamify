# Technologies

Yamify is built on a modern tech stack that emphasizes performance, developer experience, and maintainability. This page provides an overview of the core technologies used in the project.

## Frontend Framework

### Next.js

<div className="flex items-center space-x-2 mb-4">
  <img src="https://nextjs.org/static/favicon/favicon-32x32.png" alt="Next.js logo" className="w-6 h-6" />
  <span className="font-semibold">Version: 15.3.2</span>
</div>

Next.js serves as the foundation of Yamify, providing:

- **App Router**: Utilizing the latest App Router architecture for file-based routing
- **Server Components**: Leveraging React Server Components for improved performance
- **API Routes**: Handling backend functionality within the same codebase
- **Built-in Image Optimization**: Through the Next.js Image component
- **Automatic Code Splitting**: For optimized page loading

[Next.js Documentation](https://nextjs.org/docs)

## Architecture Decisions

The technology choices for Yamify were guided by these principles:

1. **Performance First**: Optimizing for user experience and load times
2. **Developer Experience**: Tools that improve productivity and code quality
3. **Type Safety**: Reducing runtime errors through static typing
4. **Component Reusability**: Designing for maximum reuse and consistency
5. **Modern Patterns**: Embracing the latest best practices in web development
